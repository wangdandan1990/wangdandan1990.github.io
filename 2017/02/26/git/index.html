1.git的使用

1)什么是git

  • 为了告别手动方式管理Linux代码,并且符合开源和免费,Linus花了 两周时间 自己用 C 写了一个分布式版本控制系统，这就是GitGit迅速成为最流行的分布式版本控制系统，
  • 尤其是 2008 年 GitHub 网站上线了，它为开源项目 免费提供Git存储 ，无数开源项目开始迁移至GitHub，包括jQuery，PHP，Ruby等等。

  • git add . (添加到暂存区)

  • git remote add origin 地址
